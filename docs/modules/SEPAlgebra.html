SEPAlgebra¶

Author: Marcel Simader

Date: 11.04.2021

class SEPModules.SEPAlgebra.AlgebraicStructure(elements: Collection[Element], binary_operators: Collection[Callable[[Element, Element], Element]])¶

AlgebraicStructure is a class designed to represent a structure \((G, \circ_1, \circ_2, \ldots, \circ_n)\) with one collection containing any number of elements of type Element and another collection of callable objects which represent the binary operators applied to the elements of \(G\).

Parameters

  • elements – a collection containing n elements of any but the same type Element, where the type given must be honored in all other uses of this instance

  • binary_operators – a collection of callable objects taking two arguments of type Element and returning one single object of type Element representing the application of the given operator onto the objects given as arguments
