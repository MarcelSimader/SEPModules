class SEPModules.SEPPrinting.Style(*styles)Â¶

A helper class to represent style and colors and which actions can be performed on them. New Style objects may be instantiated using int ANSI style codes, or the colorama.ansi classes of Colorama. Furthermore, style can be combined into composite style by the bitwise & and | and the + operator. Inplace operations are not supported as the instances of this class are immutable by design.
