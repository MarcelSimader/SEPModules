Rational¶

class SEPModules.SEPMaths.Rational(a: Union[int, float, SEPModules.SEPMaths.Rational] = 1, b: int = 1)¶

Rational number of the form \(a/b\) with \(a, b \in \mathbb{Z}\). Used for symbolic computations in SEPMaths module. Values are automatically simplified. The following forms are accepted, where option 1 and 3 are exact and option 2 will approximate the float input as integer ratio:

Parameters

  • a – may be an int, float or Rational

  • b – may be an int if a is an int, otherwise should be 1 or left out

Raises

ValueError – if the denominator b is set to 0

static __simplify__(a: int, b: int) → Tuple[int, int]¶

Takes in a tuple \((a, b)\) for integers a and b and returns a simplified tuple \((c, d)\) where \(a/b\equiv c/d\).

SEPModules.SEPMaths.find_rational_approximation(num: numbers.Real, precision: int = 4) → SEPModules.SEPMaths.Rational¶

Returns a Rational x with numerator and denominator a, b where \(\text{float}(x) = a/b \approx \text{num}\).

Parameters

  • num – the number to approximate as float or int

  • precision – the number of decimal digits of precision, if this exceeds the limits of the floating point architecture of the system, this value is truncated to this maximum

Raises

ValueError – if precision is negative

SEPModules.SEPMaths.get_possible_rationals(_set: Union[list, set]) → Set[SEPModules.SEPMaths.Rational]¶

Returns unique Rationals in a set with \(a/b \quad \forall a, b \in \text{_set}\).

Parameters

_set – all elements of _set must be integers.

Raises

ValueError – if elements of _set are not all integers
